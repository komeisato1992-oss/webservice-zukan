# 手動適用が必要なマイグレーション

ファイル:

`supabase/migrations/202607190005_comparison_display_and_contents.sql`

内容:

1. `comparison_fields` に表示箇所・比較ルール・値ソース列を追加
2. 人気3社向け初期5項目（初期費用 / 月額料金 / 無料お試し期間 / 容量 / サポート）を seed
3. `managed_contents` テーブルと RLS を追加

## 適用方法（例）

Supabase SQL Editor に上記 SQL を貼って実行するか:

```bash
supabase db push
# または
psql "$DATABASE_URL" -f supabase/migrations/202607190005_comparison_display_and_contents.sql
```

適用後、管理画面の「比較項目」で表示設定を確認し、「コンテンツ管理」で公開コンテンツを作成できます。
