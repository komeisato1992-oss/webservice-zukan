# 手動適用が必要なマイグレーション

ファイル:

`supabase/migrations/202607190006_support_comparison_fields.sql`

内容:

1. サポート関連の `comparison_fields` をカテゴリごとに seed
2. 複合表示 `support` と個別項目（電話 / メール / チャット / 対応時間など）
3. 既存の `comparison_values` は変更しない（未確認 = null のまま）

## 適用方法（例）

Supabase SQL Editor に上記 SQL を貼って実行するか:

```bash
supabase db push
# または
psql "$DATABASE_URL" -f supabase/migrations/202607190006_support_comparison_fields.sql
```

適用後、管理画面の「比較項目」および各サービスの「比較」タブでサポート情報を編集できます。
